# CloudWatch Dashboard Configuration

## Objective

Create a CloudWatch Dashboard to monitor the health and performance of EC2 instances and the Elastic Load Balancer.

---

## Dashboard Metrics

### EC2 Metrics

- CPU Utilization
- Network In
- Network Out
- Disk Read Bytes
- Disk Write Bytes

---

### ELB Metrics

- Request Count
- Target Response Time (Latency)
- Healthy Host Count
- UnHealthy Host Count

---

## Steps

1. Open AWS Management Console.
2. Navigate to CloudWatch.
3. Select Dashboards.
4. Click Create Dashboard.
5. Add widgets for EC2 and ELB metrics.
6. Save the dashboard.

---

## Result

A centralized dashboard was created to monitor system performance and resource utilization in real time.
